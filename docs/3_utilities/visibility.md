## Introduction

Visibility align utility classes.

## Classes

```html
<div class="visible">I am visible</div>
<div class="invisible">I am invisible</div>
```

## Sass mixin

```scss
.element {
  @include invisible(visible);
}
.element {
  @include invisible(hidden);
}
```
